HTML Attributes Reference

Comprehensive guide to the most commonly used HTML attributes

Global Attributes

These attributes can be used on any HTML element

Attribute Description Example
id Specifies a unique id for an element <div id="header">
class Specifies one or more class names for an element <p class="text intro">
style Specifies inline CSS styles for an element <div style="color:red;">
title Specifies extra information about an element (shown as tooltip) <a href="#" title="Home page">
data-* Used to store custom data private to the page or application <div data-user-id="123">
hidden Specifies that an element is not yet, or is no longer, relevant <div hidden>Content</div>
tabindex Specifies the tab order of an element <input tabindex="1">
contenteditable Specifies whether the content of an element is editable <div contenteditable="true">
draggable Specifies whether an element is draggable <div draggable="true">
lang Specifies the language of the element's content <div lang="en">
dir Specifies the text direction for the content in an element <div dir="rtl">
accesskey Specifies a shortcut key to activate/focus an element <button accesskey="s">Save</button>

Form Attributes

Attributes used with form elements

Attribute Description Example
action Specifies where to send the form-data when a form is submitted <form action="/submit">
method Specifies the HTTP method to use when sending form-data <form method="post">
name Specifies the name of an input element <input name="email">
value Specifies the value of an input element <input value="default">
type Specifies the type of input element <input type="email">
placeholder Specifies a short hint that describes the expected value <input placeholder="Enter email">
required Specifies that an input field must be filled out <input required>
disabled Specifies that an input element should be disabled <input disabled>
readonly Specifies that an input field is read-only <input readonly>
maxlength Specifies the maximum number of characters allowed <input maxlength="10">
min Specifies the minimum value for an input element <input type="number" min="1">
max Specifies the maximum value for an input element <input type="number" max="100">
step Specifies the legal number intervals for an input field <input type="number" step="5">
pattern Specifies a regular expression that the input's value is checked against <input pattern="[A-Za-z]{3}">
autocomplete Specifies whether a form should have autocomplete on or off <form autocomplete="off">
novalidate Specifies that the form should not be validated when submitted <form novalidate>
selected Specifies that an option should be pre-selected when the page loads <option selected>Default</option>
checked Specifies that an input element should be pre-selected <input type="checkbox" checked>
multiple Specifies that multiple options can be selected at once <select multiple>

Media Attributes

Attributes used with images, audio, and video elements

Attribute Description Example
src Specifies the URL of the media file <img src="image.jpg">
alt Specifies an alternate text for an image <img src="image.jpg" alt="Description">
width Specifies the width of the media element <img width="500">
height Specifies the height of the media element <img height="300">
controls Specifies that audio/video controls should be displayed <video controls>
autoplay Specifies that the audio/video will start playing as soon as it is ready <video autoplay>
loop Specifies that the audio/video will start over again, every time it is finished <audio loop>
muted Specifies that the audio output of the video should be muted <video muted>
poster Specifies an image to be shown while the video is downloading <video poster="image.jpg">
preload Specifies if and how the author thinks the video should be loaded when the page loads <video preload="auto">

Link and Anchor Attributes

Attributes used with <a> and <link> elements

Attribute Description Example
href Specifies the URL of the page the link goes to <a href="https://example.com">
target Specifies where to open the linked document <a target="_blank">
rel Specifies the relationship between the current document and the linked document <a rel="nofollow">
download Specifies that the target will be downloaded when a user clicks on the hyperlink <a download="filename">
hreflang Specifies the language of the linked document <a hreflang="es">

Table Attributes

Attributes used with table elements

Attribute Description Example
colspan Specifies the number of columns a table cell should span <td colspan="2">
rowspan Specifies the number of rows a table cell should span <td rowspan="2">
headers Specifies one or more header cells a cell is related to <td headers="col1">
scope Specifies whether a header cell is a header for a column, row, or group of columns or rows <th scope="col">

Meta and Script Attributes

Attributes used with <meta> and <script> elements

Attribute Description Example
charset Specifies the character encoding for the HTML document <meta charset="UTF-8">
content Gives the value associated with the http-equiv or name attribute <meta name="description" content="...">
http-equiv Provides an HTTP header for the information/value of the content attribute <meta http-equiv="refresh">
async Specifies that the script is executed asynchronously <script async>
defer Specifies that the script is executed when the page has finished parsing <script defer>
